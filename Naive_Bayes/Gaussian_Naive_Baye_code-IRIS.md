```python
# data preprocessing
x_train, x_test, y_train, y_test=train_test_split(iris.data, iris.target, test_size=0.3)
```


```python
# Gaussian Naive Bayes model
model=GaussianNB()
model.fit(x_train, y_train)
```




    GaussianNB()




```python
# test
y_pred=model.predict(x_test)
```


```python
# accuracy
accuracy_score(y_test, y_pred)
```
